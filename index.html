First-time setup

Enter your Google OAuth Client ID so this app can connect to Google Drive and read your ProofWriter documents.

Setup instructions (one-time, ~5 minutes)
  1. Go to console.cloud.google.com
  2. Create a project → APIs & Services → Library
  3. Search for and enable Google Drive API
  4. Go to Credentials → Create Credentials → OAuth 2.0 Client ID
  5. Application type: Web application
  6. Under Authorised JavaScript origins, add the URL you use to open this page (e.g. https://yourgithub.github.io or your local server address)
  7. Copy the Client ID and paste it above
  8. Folder ID: open your ProofWriter folder in Google Drive on desktop, copy the last part of the URL (e.g. drive.google.com/drive/folders/1aBcD…)
