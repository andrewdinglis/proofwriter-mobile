# ProofWriter Mobile

Mobile companion for ProofWriter — read and edit your ProofWriter documents from an Android device via Google Drive.

Open the app at: https://andrewdinglis.github.io/proofwriter-mobile/

## Setup

See the setup guide in the main ProofWriter repository.

## What this is

A single self-contained HTML page that:
- Signs in with Google OAuth
- Lists your `pw_*.json` ProofWriter documents from Google Drive
- Lets you read and edit document text on mobile
- Auto-saves changes back to Google Drive
- Optionally restricts access to a single Google account
